# bashoes

--------------------------------------
FOLDERS :
--------------------------------------
    /img : Contains all the images that are used in this website.
--------------------------------------
--------------------------------------

--------------------------------------

    /styles.css : contains the styling of the page.
    /app.js : responsible for all the features that this page provides.
  
--------------------------------------
FEATURES 
--------------------------------------
1. Colour options
2. Sizes
3. Payment form
